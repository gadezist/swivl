swivl-api

Требования:
php 8.0+
Mysql 8.0
Symfony CLI

Для запуска api сделать клон репозитория

git clone https://github.com/gadezist/swivl.git

Проинсталировать композер

php composer.phar install

Сделать миграции базы

symfony console doctrine:migrations:migrate
